# Mouse Scroll Controller

Một công cụ cho phép cuộn trang bằng các nút phụ trên chuột (X1 và X2) với tốc độ có thể điều chỉnh.

## Tính năng

- Cuộn lên/xuống bằng cách nhấn giữ nút X1 (cuộn xuống) và X2 (cuộn lên)
- Điều chỉnh tốc độ cuộn bằng thanh trượt
- Giao diện trong suốt, có thể di chuyển
- Luôn nổi trên các cửa sổ khác

## Cài đặt

1. Đảm bảo bạn đã cài đặt Python 3.7 trở lên
2. Clone repository này hoặc tải source code
3. Cài đặt các thư viện cần thiết:

```bash
pip install -r requirements.txt
